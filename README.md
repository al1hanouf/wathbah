# وثبة

## نبذة عن المشروع
وثبة هو نظام ذكي مخصص لدعم المرأة في رحلتها الصحية اليومية، من خلال تقديم توصية رياضية آمنة ومناسبة بناءً على حالتها اليومية.


## المشكلة
الكثير من النساء يواجهن صعوبة في الالتزام بخطة رياضية بسبب:
- اختلاف الحالة اليومية (تعب، نوم، دورة)
- الظروف البيئية (حر، غبار)
- عدم وجود توصيات مخصصة وآمنة


## الحل
وثبة تقدم خطة يومية ذكية تعتمد على:
- بيانات المستخدم الأساسية
- الفحص اليومي
- تحليل المخاطر
- اتخاذ قرار مناسب للحالة


## كيف يعمل النظام؟

إدخال البيانات الأساسية:
   - العمر
   - الطول
   - الوزن
   - النشاط
   - الهدف
   - عدد أيام التمرين
   - الإصابات

إدخال الفحص اليومي:
   - النوم
   - التعب
   - الدورة
   - الأعراض
   - الطقس
   - البيئة

تحليل الحالة:
   - Risk Engine لحساب مستوى الخطر

اتخاذ القرار:
   - Decision Engine لتحديد:
     - المستوى
     - الشدة
     - نوع التمرين
     - المدة

توليد التوصية:
   - باستخدام LLM

تحويل الخطة إلى صوت:
   - باستخدام TTS


## مخرجات النظام
يعرض التطبيق:

- المستوى
- نوع الخطة
- المدة
- الشدة
- التوصية اليومية
- تنبيه السلامة
- ملاحظات البيئة
- رسالة تحفيزية
- إمكانية الاستماع للخطة


## التقنيات المستخدمة

- Python
- OpenAI API
- LLM (nuha-2.0)
- Text-to-Speech (elm-tts)


## مميزات المشروع

- توصيات مخصصة يوميًا
- مراعاة الحالة الصحية
- مراعاة البيئة والطقس
- نظام تحليل مخاطر
- تجربة صوتية للخطة
